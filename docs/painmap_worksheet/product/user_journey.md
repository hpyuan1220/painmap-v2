## 1. Phase 0 — 入場

| Touchpoint | 你的狀態 | 工具行為 |
| :-- | :-- | :-- |
| 朋友轉貼 / 搜尋 / Workshop 後跟進 | 帶著一段模糊的卡住感進來 | — |
| 入口頁 `/learn/worksheet` | 想知道「這要花我多久、適不適合我」 | 用三句話自我介紹：「一本陪你做質性研究的筆記本 / 30–90 分鐘 / 不需要懂 AI」 |
| 點擊「開始一段新的探索」 | 願意慢下來 | 1. 檢查 LocalStorage 是否有未完成的 PainCard<br>2. 有 → 「歡迎回來，我們上次走到 Card N，要繼續嗎？」<br>3. 無 → 建立新 PainCard，跳轉 `/learn/worksheet/01` |

**情緒**：好奇 / 略帶懷疑（「這是不是又一個 idea 評分器？」）

**入口文案範例**（符合 `voice_and_tone.md`）：

> 一本陪你做質性研究的筆記本。
>
> 我們不打分數、不評等、不替你判斷生意能不能做。
> 我們只想陪你把一段卡住的故事慢慢聽清楚，
> 一直走到你準備好去找真人聊一聊為止。

## 2. 13 張卡片詳細旅程

### Card 1 · 那句脫口而出的話

| 項目 | 內容 |
| :-- | :-- |
| 路徑 | `/learn/worksheet/01` |
| 對應欄位 | `complaint.*` |
| AI 介入 | ❌ |
| 主要動作 | 把最近聽到（或自己脫口而出）的抱怨原話寫下來，附上誰說、何時、何地、當時做什麼 |

**情緒**：好奇 → 慢下來

**走下一張卡前，我們想多聽你說的事**：
- 這句話是誰說的？（一個你叫得出名字的人）
- 大概什麼時候、什麼場景下說的？

如果還寫不出來，工具會說：「沒關係，這還不是你的題目，下次聽到一句具體的抱怨再回來找我們。」

### Card A · 痛點現場日記 *(NEW)*

| 項目 | 內容 |
| :-- | :-- |
| 路徑 | `/learn/worksheet/02` |
| 對應欄位 | `pain_diary.entries[]` |
| AI 介入 | 選擇性（使用者可選擇要不要請 AI 看看這幾段日記） |
| 主要動作 | 在接下來的 3–7 天，記下 3–5 個跟這個抱怨有關的當下時刻：時間、地點、心情、觸發了什麼、自由書寫一句話 |

**為什麼這張卡**：2026 質性研究強調 mobile ethnography — 在事情發生的當下捕捉一手感受，比事後回憶準確。

**情緒**：被打開、開始注意自己

**走下一張卡前**：
- 至少 1 段現場日記（建議 3 段）

工具的語氣：

> 接下來幾天，當這個卡住的感覺又冒出來時，
> 隨手寫一兩句進來就好。
> 不用整理、不用美化，誠實的當下最有用。

### Card 1-A · AI 替你打開三條路

| 項目 | 內容 |
| :-- | :-- |
| 路徑 | `/learn/worksheet/03` |
| 對應欄位 | `ai_narrowing.directions[]` |
| AI 介入 | ✅ |
| 主要動作 | 複製一段邀請式 prompt 給 AI（外部 ChatGPT / Claude / Perplexity / Gemini），請 AI 從你的抱怨原話 + 日記出發，給你 3 條值得繼續往下聽的方向 |

**為什麼這張卡**：一段抱怨通常藏著 2-4 條不同的痛點線。直接收斂會錯過真正重要的那條。

**情緒**：被打開、有點驚訝

**走下一張卡前**：
- AI 回了 3 條方向
- 你選了其中一條（不刪除其他兩條，只是這次先走這條）

工具的語氣：

> 把你寫的抱怨 + 日記貼給 AI，
> 不是要它替你想解法，而是請它替你打開幾條你可能還沒注意到的方向。
> 三條路裡，你最想再多聽哪一條？

---

### Card 1-B · 走進其中一條，慢慢往下問

| 項目 | 內容 |
| :-- | :-- |
| 路徑 | `/learn/worksheet/04` |
| 對應欄位 | `ai_narrowing.drill_rounds[]` |
| AI 介入 | ✅ |
| 主要動作 | 跟 AI 進行 2–3 輪「往下問」。每輪你寫一個問題、AI 回應、你寫一段反思。每一輪都不能要 AI 給解法 |

**為什麼這張卡**：選了方向之後，需要持續向下追問才能逼出真正的痛點層次。

**情緒**：被打開 → 開始收斂

**走下一張卡前**：
- 至少 2 輪 drill（建議 3 輪）
- 每輪都有你自己的反思

工具的語氣：

> 我們選了一條路，現在開始往下走。
> 每一輪，你問一個更具體的問題，AI 陪你看一看，
> 然後你寫一句「我聽到了什麼」就好。

### Card B · 心情地圖 *(NEW)*

| 項目 | 內容 |
| :-- | :-- |
| 路徑 | `/learn/worksheet/06` |
| 對應欄位 | `empathy_map.*` |
| AI 介入 | 選擇性 |
| 主要動作 | 填 Think / Feel / Say / Do / Pain / Gain 六個欄位，站在那個有名字的人的位置 |

**為什麼這張卡**：2026 質性研究強調 empathy-driven understanding — 在進入「公式」「解法」之前，先真正進入那個人的心情。

**情緒**：進入心情、開始有溫度

**走下一張卡前**：
- 六個欄位皆非空

工具的語氣：

> 我們站到那個人的位置上看看。
> 他這個時候心裡在想什麼？身體上、表情上會出現什麼？
> 嘴上會說什麼、跟心裡想的一樣嗎？

### Card 4 · 把卡點輕輕說清楚 + AI 解法回看

| 項目 | 內容 |
| :-- | :-- |
| 路徑 | `/learn/worksheet/07` |
| 對應欄位 | `stuck_formula_with_solutions.*` |
| AI 介入 | ✅ |
| 主要動作 | 寫「我每次要 ___，都會卡在 ___」的句子；請 AI 列出常見解法；逐一回看每個解法，寫為什麼這個解法不夠（v2 的關鍵反 solution-mode 練習）|

**情緒**：落地、開始判斷

**走下一張卡前**：
- 句子非空
- 至少對 3 個 AI 解法寫了「為什麼這個不夠」

工具的語氣（AI 提解法時）：

> 這是市場上常見的幾個解法。
> 我們不急著評論它好壞，
> 只想請你誠實寫一寫：如果用這個，你心裡那個卡住的感覺，會不會就消失？

### Card 6 · 市場聲音的三段證據

### Card 7 · 三個有名字的人 + 你心裡的猜想

| 項目 | 內容 |
| :-- | :-- |
| 路徑 | `/learn/worksheet/10` |
| 對應欄位 | `people_with_guesses.*` |
| AI 介入 | 選擇性 |
| 主要動作 | 寫 3 個你叫得出名字 + 聯絡得到的人；每人寫「為什麼是他」+ 5 個你預先猜他會給的答案 |

**為什麼這張卡**：2026 質性研究核心 — 你必須先寫下你的預期，才能在訪談中辨認「驚訝」。沒有預期就沒有發現。

**情緒**：看見人、有期待也有不安

**走下一張卡前**：
- list.length === 3
- 每人至少 3 個 guessed_answers（建議 5）

### Card D · 自我假設清單 *(NEW)*

| 項目 | 內容 |
| :-- | :-- |
| 路徑 | `/learn/worksheet/11` |
| 對應欄位 | `assumptions.*` |
| AI 介入 | ❌（這張要你自己寫，不請 AI 看） |
| 主要動作 | 列出你目前心裡的假設 + 手上的證據 + 「訪談中要聽到什麼，我才會修正這個假設」 |

**為什麼這張卡**：2026 質性研究強調 reflexivity — 訪談前先盤點偏見，否則容易引導受訪者。

**情緒**：自我檢查、有點不舒服

**走下一張卡前**：
- 至少 2 個 assumptions
- `biases_to_watch` 非空

工具的語氣：

> 走進對話前，我們先把自己心裡的猜想攤開來看一看。
> 不是要你放棄這些猜想，
> 是要你記得：等一下對方說的話如果跟你的猜想不一樣，
> 不要急著解釋掉它。

## 4. 中斷與回來

任何卡片皆可暫停。回來時：
- LocalStorage 自動還原到上次離開的那張卡
- 看到一段歡迎訊息（見 `voice_and_tone.md` §6.3）
- 任何已寫過的內容都不會被清空
- 可以隨時回到任何前面的卡片補充
